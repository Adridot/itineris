Itineris Privacy Policy

Last updated: February 12, 2026

This Privacy Policy explains how the Itineris Chrome extension ("Itineris", "we", "our") handles user data.

1. What Itineris does

Itineris compares travel times from one origin address to multiple saved destinations, based on selected transport mode and optional departure or arrival time.

2. Data we process

To provide the core functionality, Itineris processes the following user-provided data:

  • Origin addresses
  • Saved destination names and addresses
  • Favorite origins and origin history
  • User preferences (transport mode, time options, filters, cache settings)
  • Route request parameters required to compute travel times

3. Where data is stored

  • Local and synced extension storage: Itineris stores destinations, preferences, favorites, and recent results in Chrome storage (`chrome.storage.sync` and `chrome.storage.local`).
  • Backend processing: When you request travel times, the extension sends request data to our backend endpoint hosted on AWS Lambda, which then queries Google Maps routing services to return results.

4. How data is used

Data is used only to provide and improve the extension's route-comparison functionality, including:

  • Computing travel times for your requested routes
  • Displaying sorted route results in the extension UI
  • Remembering your saved destinations and preferences
  • Reducing repeated requests through local caching

5. Data sharing

We do not sell personal data. We do not use your data for advertising or credit-scoring purposes. Data is shared only as needed to provide the core service:

  • AWS (hosting and backend execution)
  • Google Maps routing APIs (route computation)

6. Permissions used by the extension

  • storage: to save destinations, preferences, favorites, and cache entries
  • host access: limited to the declared backend API host used to request route data

7. Remote code

Itineris does not execute remote JavaScript or remote Wasm code. Executable code is packaged with the extension.

8. Your controls

You can control and remove your data at any time by:

  • Editing or deleting destinations in the Options page
  • Clearing favorites and cache from the Options page
  • Removing the extension from Chrome

9. Policy updates

We may update this Privacy Policy to reflect product or compliance changes. The "Last updated" date will be revised when updates are made.

10. Contact

For privacy questions, please contact us via: https://github.com/Adridot/itineris/issues
